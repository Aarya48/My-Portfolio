# Personal Portfolio

A modern and responsive developer portfolio built with React. It showcases my projects, skills, experience, and contact information in a clean and interactive interface.

## Tech Stack
- React.js
- JavaScript
- Tailwind CSS
- Vite

## Features
- Responsive Design
- Project Showcase
- Interactive UI
- Contact Form
- Smooth Animations

## Installation

```bash
git clone <repository-url>
cd portfolio
npm install
npm run dev
